# WithPano
[WithPano](https://github.com/arhazra/withpano) Panoramic application/builder app, built with [Marzipano](https://github.com/google/marzipano)


# Development

## System Requirements

- NodeJS
- Python3.9
- React

Run local server
```
$ python -m withpano
```
Browse to http://127.0.0.1:8090
